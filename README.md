# cc-reactor-monitor
Extreme Reactors monitoring software for operators who enjoy knowing whether their reactor is about to run out of fuel.

Developed for ATM10.

## Requirements

- ComputerCraft computer
- Monitor
- Extreme Reactors computer port

The computer must be directly connected to both the monitor and the reactor computer port.

## Features

- Displays:
  - Stored energy
  - Average power generation
  - Fuel level
  - Waste amount
  - Fuel efficiency (FE/mB)
- Outputs a redstone signal (strength 15) on the top side when fuel drops below 90%.
